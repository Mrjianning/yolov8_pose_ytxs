
## 引体向上动作分析系统

### 1. 项目介绍

本项目是一个基于深度学习的动作分析系统，旨在通过分析人的动作来识别不同的动作类型。该系统使用了深度学习模型来对动作进行分类，并且提供了一个简单易用的用户界面。

### 2. 项目结构

-`data`: 存放数据

-`models`: 存放模型

### 3. 环境要求

- Python 3.8+
- PyTorch 1.7+
- OpenCV

### 4. 安装依赖

```

<!-- 创建conda环境 -->

conda create -n pose python=3.8


<!-- 激活环境 -->

conda activate pose


pip install -r requirements.txt  -i https://mirrors.aliyun.com/pypi/simple

```
